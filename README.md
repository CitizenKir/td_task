# Тестовое задание TradeDealer

Задание сделано на Laravel

## Запуск проекта

1. **Скопировать переменные окружения:**
   ```bash
   cp .env.example .env

2. **Запустить контейнеры:**
   ```bash
   sail up

3. **Выполнить миграцию:**
   ```bash
   sail artisan migrate

4. **Заполнить базу:**
   ```bash
   sail artisan db:seed
